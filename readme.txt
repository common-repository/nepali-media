Links to popular Nepali media


== Description ==

Links to popular Nepali media cited from [w3newspapers](https://www.w3newspapers.com/nepal/) and other resources.


== Installation ==

Search and activate the plugin named: "nepali media".


== Frequently Asked Questions ==

= Is SQLite3 necessary?
Yes.

= Does it add something to the database?
No. Rather it contains the list of media in an embeded database using SQLite3.
